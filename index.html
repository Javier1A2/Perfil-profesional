Estación SatNOGS

Proyecto enfocado en la configuración de una Raspberry Pi para recibir y procesar datos de satélites con fines de investigación. La estación funciona como parte de la red colaborativa SatNOGS, que integra estaciones alrededor del mundo para la observación y análisis de señales. Este proyecto me permitió aprender sobre comunicaciones satelitales, radiofrecuencia y automatización de sistemas.

Tesis "Brecha Digital"

Investigación académica sobre la brecha digital y su impacto en la inserción laboral, con un enfoque particular en las MiPyMEs de Colima, México. Se analiza cómo el acceso desigual a internet, dispositivos y competencias digitales influye en las oportunidades de crecimiento empresarial y empleabilidad. El trabajo incluye un marco teórico, estudios de caso locales y una propuesta de estrategias para reducir esta problemática mediante capacitación y políticas públicas enfocadas en la digitalización.

Juego "Jump Man"

Videojuego 2D desarrollado en equipo para el microcontrolador ESP32. Incluye una pantalla LCD, joystick y salida de audio mediante protocolo I2S. La arquitectura implementa el uso de doble núcleo: uno destinado a la lógica del juego (movimiento, colisiones y puntuaciones) y otro para la gestión del audio. Fue un proyecto retador que combinó programación de bajo nivel con creatividad en el diseño del juego.

Aplicación móvil con Expo

Aplicación móvil multiplataforma desarrollada con React Native y Expo, compatible con Android, iOS y Web. El entorno se configuró mediante create-expo-app, integrando un sistema de navegación basado en archivos y compatibilidad con Expo Go para pruebas rápidas. Este proyecto permitió comprender el desarrollo ágil de apps móviles y la importancia de contar con herramientas que faciliten la portabilidad.
